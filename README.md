# Cool-X Messaging Prototype

A modern, high-performance messaging app prototype inspired by Telegram, built as a responsive single-page experience for Web (with mobile-first UX suitable for Android/iOS design adaptation).

## Highlights
- Glassmorphism + flat UI hybrid with smooth gradients and rounded cards.
- Light/Dark mode toggle.
- Splash + OTP login flow.
- Bottom navigation: Chats, Groups, Channels, Calls, Settings.
- Interactive chat simulation with typing indicators, reactions, send animation, and delivered ticks.
- Group/Channel creation dialog and dynamic cards.
- Global chat search and responsive layout.

## Run locally
Open `index.html` directly in a browser, or run a static server:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.
